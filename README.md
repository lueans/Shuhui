#### 主要运用
- `Material Design`
- [Retrofit2](https://square.github.io/retrofit/) RESTful API库与响应式编程[Rxjava](https://github.com/ReactiveX/RxJava)结合
- [DBflow](https://github.com/Raizlabs/DBFlow)数据库
- [Glide](https://github.com/bumptech/glide) 图片加载
